## Что нужно сделать после установки

1. **Выполните SQL** из `SUPABASE_SQL.md` в Supabase SQL Editor
2. **Обновите `.env`** — добавьте `NEXT_PUBLIC_APP_URL` для Mini App
3. **Настройте webhook бота:**
   ```bash
   curl https://api.telegram.org/bot<YOUR_TOKEN>/setWebhook?url=https://your-site.vercel.app/api/telegram/webhook
   ```
4. **В админке → Настройки** заполните все поля
5. **Готово!** Всё работает
