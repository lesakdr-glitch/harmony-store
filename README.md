# Harmony Store

Интернет-магазин продукции Vilavi для митохондриального здоровья.

## Технологии

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Supabase
- Telegram Bot API
- Recharts

## Установка

1. Скопируйте `.env.example` в `.env` и заполните переменные окружения
2. Установите зависимости: `npm install`
3. Запустите dev сервер: `npm run dev`

## Настройка Supabase

1. Создайте проект в Supabase
2. Выполните SQL из файла `supabase_schema.sql`
3. Выполните SQL из файла `supabase_rls.sql`

## Деплой

Проект готов к деплою на Vercel.
