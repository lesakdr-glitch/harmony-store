# Harmony Store — магазин продукции Vilavi

Элегантный онлайн-магазин продукции Vilavi для митохондриального здоровья. Природный минимализм, Apple-дизайн, три уровня доступа (админ, продавец, покупатель).

## ✨ Особенности

### 🎨 Дизайн
- **Цвета:** бежево-кремовые тона (#FAF7F2 фон, #8B7355 коричневый акцент, #6B8E4E оливковый)
- **Шрифты:** Raleway (заголовки), Inter (текст)
- **Стиль:** природный минимализм, Apple-эстетика
- **Параллакс:** фон с анимацией при скролле
- **Аудитория 40+:** крупный шрифт (16-18px), понятная навигация

### 🛒 Функционал
- **Покупатель:** регистрация, корзина, избранное, заказы, личный кабинет
- **Продавец:** управление своими категориями/товарами, заказы
- **Админ:** полный контроль над сайтом и пользователями
- **Telegram бот:** /start, /orders, уведомления
- **Отслеживание:** /track по номеру телефона

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка Supabase
1. Создайте проект на [supabase.com](https://supabase.com)
2. Скопируйте `Project URL` и `anon public key` в `.env`
3. Выполните SQL из `supabase_schema.sql`
4. Выполните RLS политики из `supabase_rls.sql`

### 3. Настройка переменных окружения
```bash
cp .env.example .env
# Заполните все значения в .env
```

### 4. Запуск разработки
```bash
npm run dev
```
Откройте [http://localhost:3000](http://localhost:3000)

## 🗄️ Структура базы данных

### Таблицы
1. **users** — пользователи (admin/seller/customer)
2. **categories** — категории товаров
3. **products** — товары (с ссылками на Ozon/WB/Yandex)
4. **orders** — заказы с JSONB товарами
5. **favorites** — избранное пользователей
6. **reviews** — отзывы
7. **settings** — настройки сайта

### Роли доступа
- **Админ:** Всё управление
- **Продавец:** Свои товары/категории/заказы  
- **Покупатель:** Покупки, избранное, отзывы

## 🔒 Безопасность

### Row Level Security (RLS)
- Каждая таблица защищена RLS политиками
- Пользователи видят только свои данные
- Продавцы — только свои товары и заказы
- Админы — всё

### Авторизация
- iron-session с HttpOnly куками
- bcrypt хеширование паролей
- JWT для API запросов
- Проверка ролей на всех уровнях

## 🤖 Telegram бот

### Команды
- `/start` — приветствие + кнопки (Каталог, Мои заказы, Поддержка, FAQ)
- `/orders` — последние заказы (для админа/продавца)

### Уведомления
- Новые заказы отправляются админу и продавцу
- Webhook автоматически устанавливается при первом запросе

## 💳 Оплата и доставка

### Оплата
- **СБП (QR-код)** — мгновенная оплата
- **Наложенный платёж** — при получении

### Доставка
- **СДЭК** — 3-7 дней по России
- **Самовывоз** — только Новороссийск


### Docker (опционально)
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["npm", "start"]
```
